# SmartPark AI

A web-based Smart Parking Availability & Prediction System developed by PHILIP AGANO AULO as a final year BSc (Hons) Computing project.

## Project Objectives

- Monitor parking availability in real time
- Reserve parking slots
- Predict future parking occupancy using Machine Learning
- Provide dashboards for drivers, operators and administrators

## Technology Stack

### Backend
